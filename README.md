# 📡 RSMA vs OMA Sum Rate Simulation (MATLAB)

> **A MATLAB simulation comparing the performance of Rate-Splitting Multiple Access (RSMA), Optimized RSMA, and Orthogonal Multiple Access (OMA) over Rayleigh fading channels.**

## ✨ Highlights

* 📈 Sum Rate vs. SNR Analysis
* ⚡ Optimized RSMA with Power Allocation Search
* 📶 Rayleigh Fading Channel Model
* 🎲 10,000 Monte Carlo Simulations
* 🔬 Ideal for 5G/6G Wireless Communication Research

## 🚀 Getting Started

1. Clone this repository.
2. Open `Sum_rate_for_RSMA.m` in MATLAB.
3. Run the script.

The simulation automatically computes and compares:

* ✅ RSMA
* ✅ Optimized RSMA
* ✅ OMA

and generates the **Sum Rate vs. SNR** performance plot.

## 📊 Simulation Setup

| Parameter        | Value           |
| ---------------- | --------------- |
| Channel          | Rayleigh Fading |
| SNR Range        | 0–30 dB         |
| Monte Carlo Runs | 10,000          |
| Users            | 2               |
